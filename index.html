    How This MVP Works

    This GitHub Pages app runs a browser-based version of the Helios recommendation engine. It mirrors the prototype’s evapotranspiration, stress scoring, and irrigation optimization logic so growers can use it instantly without a backend.

    For production, this frontend can connect to the FastAPI backend and the trained LightGBM model already included in the repository.
